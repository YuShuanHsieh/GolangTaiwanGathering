一般來說，我們都認為 Golang 的 concurrency primitives， 能讓我們更好也更容易的去處理 concurrency program，但這篇論文 Understanding Real-World Concurrency Bugs in Go 提供了不同的看法，而我們也可以從中學習到一些 Golang 特有的 concurrency bugs.
